Graphs and Graph Models

Discrete Mathematical Structures - Unit 3

A graph model is a mathematical structure used to represent relationships or connections between objects. It consists of vertices (also known as nodes) that represent individual entities and edges (also called arcs or links) that represent the relationships between these entities.

Types of graphs:

  • Simple Graph: A graph where no self-loops (an edge connecting a vertex to itself) or multiple edges between the same pair of vertices are allowed.
  • Multigraph: A multigraph is a type of graph that allows multiple edges between the same pair of vertices. In other words, it permits parallel edges or multiple connections between two vertices.
  • Pseudograph: A pseudograph is a type of graph that allows self-loops, which are edges that connect a vertex to itself. It may also allow multiple edges between the same pair of vertices, similar to a multigraph.
  • Directed Graph (Digraph): A graph where each edge has a specific direction, indicated by arrows. The connections between vertices are one-way.
  • Complete Graph: A graph where every pair of distinct vertices is connected by an edge. In a complete graph with n vertices, there are n(n-1)/2 edges.

Graph models have numerous practical applications. They are used in social network analysis to study relationships among individuals, in routing algorithms to find the most efficient paths in computer networks, in recommendation systems to suggest relevant items based on user preferences, and in many other domains.
