Expected Value and Variance

Discrete Mathematical Structures - Unit 2

Expected Value

Expected value is a concept used in probability and statistics to measure the average outcome of a random event over the long run. It provides a way to estimate what you might expect to gain or lose on average when you repeat an experiment or take part in a random process multiple times.

In simple terms, the expected value is the sum of the possible outcomes of an event, weighted by their respective probabilities. It represents the average value you can expect to obtain over many trials.

To illustrate this, let's consider rolling a fair six-sided die. Each face of the die has a number from 1 to 6. The expected value of a single roll can be calculated by taking the sum of each outcome multiplied by its probability.

For this example, the probability of rolling a 1, 2, 3, 4, 5, or 6 is 1/6 each because the die is fair.

So, the expected value is: (1/6) * 1 + (1/6) * 2 + (1/6) * 3 + (1/6) * 4 + (1/6) * 5 + (1/6) * 6 = 3.5

Therefore, the expected value of a single roll of a fair six-sided die is 3.5. This means that, on average, you can expect the outcome of rolling the die repeatedly to converge towards 3.5.

It provides a useful tool for assessing risk and estimating long-term average results.

Variance

Variance is a measure of how spread out or dispersed a set of data or probability distribution is. It quantifies the variability or deviation from the expected value. In the context of probability and statistics, variance is often used to understand the degree of uncertainty or variability associated with a random variable or a set of observations. It tells us how much the values in a data set differ from the average.

Let's consider a random variable X that follows a discrete probability distribution with outcomes X1, X2, ..., Xn, each occurring with probabilities p1, p2, ..., pn, respectively.

The variance (Var) of this discrete probability distribution can be calculated using the formula:
Var(X) = (X1 - μ)2 * p1 + (X2 - μ)2 * p2 + ... + (Xn - μ)2 * pn,
where μ is the mean or expected value of X, given by:
μ = X1 * p1 + X2 * p2 + ... + Xn * pn.

A large variance indicates that the data points are more spread out, meaning they deviate from the mean to a greater extent. Conversely, a small variance suggests that the data points are closer to the mean, indicating less dispersion.

It is widely used in various statistical analyses, such as hypothesis testing, decision-making, and modeling, to understand the distribution of data and make inferences about populations based on samples.
